# CUSTOMER-SHOPPING-BEHAVIOUR-ANALYSIS

## 📌 Project Overview

This project performs **customer trend analysis** using a combination of **SQL, Python, and Power BI** to extract insights from transactional data.

The objective of this project is to analyze customer purchasing behavior, identify patterns, and generate business insights that can help companies improve **marketing strategies, customer retention, and revenue growth**.

The workflow includes:

- Data cleaning and preprocessing
- SQL-based exploratory analysis
- Python-based data analysis
- Interactive dashboard creation using Power BI

This project demonstrates how **data engineering, data analysis, and visualization** can be combined to derive meaningful insights from raw data.

---

# 🧠 Key Objectives

The project focuses on answering the following questions:

- What are the overall **customer purchasing trends**?
- Which **products or categories generate the most revenue**?
- Who are the **most valuable customers**?
- How frequently do customers make purchases?
- What patterns exist in **customer spending behavior**?

# 📊 Data Analysis Workflow

The analysis follows a structured pipeline:

### 1️⃣ Data Collection
Customer transaction data is imported into the system.

### 2️⃣ Data Cleaning
Python is used to clean the dataset by:

- Removing missing values
- Formatting columns
- Handling inconsistent data

### 3️⃣ SQL Analysis
SQL queries are used to extract insights such as:

- Total revenue
- Customer purchase frequency
- Top selling products
- Customer segmentation

### 4️⃣ Data Visualization
Power BI dashboards visualize key insights including:

- Revenue trends
- Customer distribution
- Product performance
- Customer segmentation

# 🚀 How to Run This Project

1. **Clone the repository**
   ```bash
   git clone https://github.com/mayankm2105/CUSTOMER-SHOPPING-BEHAVIOUR-ANALYSIS
   cd customer-trends-data-analysis-SQL-Python-PowerBI
   ```
2. **Open Customer_Shopping_Behavior_Analysis.ipynb notebook**

    This file contains:

      - Data Import

      - Data exploration

      - Data cleaning

      - Connection to SQL Database
  
3. **Load the data from Python notebook into MySQL/PostgreSQL/MS SQL Server**

      - Create a database in SQL

      - Run Python code to load data into SQL database
  
      - Open **customer_behavior_sql_queries.sql**
  
      - Answer Business Questions using SQL Queries 
      
4. **Connect the SQL Database to Power BI**

      - Open **customer_behavior_dashboard.pbix**
   
      - Create interactive dashboard in Power BI
  
6. **Create Project Report and Presentation**

      - Create project report
   
      - Build presentation deck using Gamma AI
